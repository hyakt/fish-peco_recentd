# peco_recentd

move to recent directory with peco.

## Install

With [fisherman]

With [z]

```
fisher tsu-nera/fish-peco_recentd
```

## Usage
Please bind peco_recentd your favorite key.

If you'd like to bind CTRL+x CTRL+r

```fish
function fish_user_key_bindings
  bind \cx\cr peco_recentd
end
```
